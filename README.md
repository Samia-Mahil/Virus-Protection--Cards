---

### 📋 وصف المشروع

بطاقات تعليمية تفاعلية مصممة خصيصاً لطلاب مدرسة مدينة الشامخة لتعليمهم كيفية حماية أجهزتهم المدرسية من الفيروسات والمخاطر الرقمية.

---

### ✨ المميزات

- 🃏 **10 بطاقات تفاعلية** قابلة للقلب (Flip Cards)
- 📚 **+50 نصيحة أمنية** موزعة على 5 فئات
- 🏆 **نظام تتبع التعلم** مع شريط تقدم
- 🎯 **فلترة حسب الفئة** (وقاية، برامج، شبكة، بيانات، سلوك)
- 🎊 **شاشة إنجاز** مع كونفيتي عند إكمال جميع البطاقات
- 📱 **متوافق مع الهاتف** ومُحسَّن لتطبيق تيليغرام
- 🌐 **ملف واحد مكتفٍ بذاته** — لا يحتاج إنترنت بعد التحميل

### 🚀 كيفية النشر على GitHub Pages

```bash
# 1. أنشئ repository جديد على GitHub
# 2. ارفع ملف index.html (أعد تسميته من virus_protection_cards.html)
# 3. اذهب إلى Settings > Pages
# 4. اختر Branch: main و Root folder
# 5. احفظ — سيكون الرابط جاهزاً خلال دقائق
```

**أو استخدام GitHub CLI:**
```bash
git init
git add .
git commit -m "Add virus protection interactive cards"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```
